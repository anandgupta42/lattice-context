60-Second Quickstart

# 1. Install pip install lattice-context # 2. Initialize in your dbt project cd your-dbt-project lattice init # 3. Index your project lattice index # 4. Start the MCP server lattice serve

Then add to Claude Desktop config:

{ "mcpServers": { "lattice": { "command": "lattice", "args": ["serve"] } } }

Restart Claude Desktop. Now it knows your project's context! üéâ
